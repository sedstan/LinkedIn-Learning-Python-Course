# Stuctured Data

## The list type

`x = [1, 2, 3, 4, 5 ]`
This is mutable

## Tuple type

`x = ( 1, 2, 3, 4, 5 )`
This is immutable

## Dictionary type

`x = { 'a': 1, 'b': 2, 'c': 3 }`
This is a sequence of key value pair

## Set type

`x = { 1, 2, 3, 4 }`
A unordered list of unique values within a sequence. Useful to compare common values in multiple sets
