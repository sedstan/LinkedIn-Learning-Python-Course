# Operators

## Bitwise Operators

`&`        And
`|`        Or
`^`        Xor
`<<`       Shift left
`>>`       Shift right

operate on numbers and individual bits in the numbers

## Comparison Operators

`<`       Less than
`>`       Greater than
`<=`      Less than or equal
`>=`      Greater than or equal
`==`      Equal
`!=`      Not equal

## Boolean Operators

`and`     And
`or`      Or
`not`     Not
`in`      Value in set
`not in`  Value not in set
`is`      Same object identity
`is not`  Not same object identity
